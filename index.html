Keyboard Navigation Controls

BrainSprint supports full accessibility and fluid speedruns via keybindings. You can play the entire quiz without touching your mouse!

Key Action Triggered
1, 2, 3, 4 Select Option A, B, C, or D (or index 1-4) respectively
A, B, C, D Select Option A, B, C, or D respectively (alternative mapping)
Enter or Space Confirm selected answer / Proceed to next question
S or ArrowRight Skip the current question (when answer has not been locked)
ArrowLeft or ArrowRight Scroll through review items on the results screen
Esc Quit active quiz run or close this helper modal
M Toggle sound effects mute/unmute
T Toggle Dark / Light color theme
Dynamic synthesized audio activated! Enjoy retro soundscapes.
