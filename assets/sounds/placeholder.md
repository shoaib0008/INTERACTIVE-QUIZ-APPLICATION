# Sounds Assets Directory

This folder is designed to store external sound files (like `.mp3` or `.wav`) if you wish to expand the sound system in the future.

Currently, **BrainSprint** utilizes a highly advanced, zero-dependency **Web Audio API Sound Synthesizer** written in JavaScript (`script.js`). This synthesizer generates high-quality retro chime sound waves dynamically in the browser, eliminating the need to load external media files.
